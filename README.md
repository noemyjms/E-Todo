# E-Todo
*/Projet_Epitech/Bachlor1/Web* 

**Status :** Premier projet Web réalisé après la piscine Web Full‑Stack.

**Objectif :** Créer une application complète comprenant :
- une interface de présentation,
- une page d’inscription,
- une page de connexion,
- une application permettant la création, la suppression et la modification de todolists.

**Langages utilisés :**
- HTML
- CSS
- Javascript
- SQL

**Outils utilisés :**
- Node.js
- Express
- Docker

**Réalisation :** 

Ce projet était notre première vraie plongée dans le développement Web, du front au back. On a commencé par construire une interface simple, avec une inscription, une connexion, puis un espace où l’utilisateur peut gérer sa todolist : créer une tâche, la modifier, la supprimer, lui ajouter une deadline et un statut pour mieux s’organiser.

Au fil de l’avancement, on a voulu rendre l’application un peu plus vivante. On a ajouté la possibilité de barrer une tâche une fois terminée, quelques animations pour rendre l’utilisation plus agréable, et une interface pensée pour rester claire et accessible.

Malheureusement, avec le temps imparti, on n’a pas pu intégrer toutes les idées prévues. Trois boutons sont encore inactifs pour le moment. Peut‑être qu’ils trouveront leur utilité plus tard, si on décide de revenir sur le projet et de le pousser un peu plus loin.
